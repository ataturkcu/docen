# Welcome to Docen

Docen is a minimal, developer-friendly document page engine. 
It uses pure HTML, CSS, and JavaScript.

## Features

- **No Frameworks:** Built with pure vanilla JS.
- **Easy Config:** Add pages in `js/config.js` and ship.
- **Hash Routing:** Works on static hosting without backend routing.
- **Safe Markdown Render:** Markdown output is sanitized before render.
- **Search:** Fast search with lazy indexing and in-memory cache.
- **On-page TOC:** Desktop right panel + mobile bottom-sheet TOC.
- **Page Navigation:** Previous/Next links at the bottom of each page.
- **Theme Support:** Light and dark theme with saved preference.
- **Keyboard Shortcuts:** `/`, `n`, `p`, `Esc`.

## Notice

Docen uses **Lucide Icons** (`https://lucide.dev`) for UI icons.
