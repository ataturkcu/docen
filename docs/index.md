# Welcome to Docen

Docen is a minimal, developer-friendly document page engine. 
It uses pure HTML, CSS, and JavaScript.

## Features

- **No Frameworks:** Built with pure vanilla JS.
- **Easy Config:** Add your pages to `config.js` and you are good to go.
- **Customizable Theme:** Native CSS variables to shape colors, fonts, shapes.
- **Hash-routing:** Works statically without backend routing setups.
- **Markdown parsing:** Full markdown properties are parsed and themed!
